Flash the firmware straight from your browser — no toolchain, no PlatformIO.

Use Chrome or Edge on desktop (Web Serial required). Serve this page over HTTPS to flash.
Installs firmware v__FWVER__
  1. Get a free aisstream.io API key (non-commercial) at aisstream.io — you'll paste it in step 4.
  2. Plug the board into your computer with a USB-C data cable.
  3. Click Install, pick the USB JTAG/serial (or USB Serial) port, then Erase device + Install on first flash.
  4. When it reboots, join the CapsuleMarine-Setup WiFi from your phone to enter your network, aisstream API key, and home location.
  5. Fine-tune anything later at http://capsule-marine.local/.
